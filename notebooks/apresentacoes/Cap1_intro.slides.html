Descrição do Problema¶

Uma das causas desse problema é que há uma altíssima barreira de entrada para a pesquisa aplicada nessa área, são elas:

  1. A primeira barreira seria dominar todas as áreas relacionadas e necessárias para elaborar uma estratégia de investimentos.
  2. Restrições orçamentárias limitam o tamanho de uma equipe. Isso é um problema, porque uma equipe enxuta de pesquisadores tem amplitude limitada.
  3. Direitos de confidencialidade do provedor dados impede que esses dados sejam disponibilidados publicamente.
  4. Incentivo a falsos positivos. Pesquisadores são remunerados para produzir simulações de estratégias de investimentos com resultados promissores em um conjunto de dados limitado, levando a um viés de seleção (overfitting).

Fonte: DE PRADO, M. L., FABOZZI, F. J. “Crowdsourced Investment Research Through Tournaments”, The Journal of Financial Data Science, v. 2, n. 1, pp. 86–93, 2020.

Uma Alternativa para o problema¶

De Prado (2020), sugere a formação de um torneio, onde um organizador formaliza um desafio de estratégias investimento em termos de um problema preditivo, para incentivar a pesquisa de maneira coletiva.

Os torneios dividem o problema de estratégias investimento em duas tarefas especializadas:

  • i. Estruturar os dados de forma a constituir um problema preditivo, que é executada pelos organizadores do torneio, que possuem extenso conhecimento em finanças;

  • ii. Modelagem, que é feita pelos participantes do torneio com conhecimento de ciência de dados.

Como Suplantar as barreiras através do torneio¶

  1. Como o problema de previsão já foi definido os participantes do torneio podem trabalhar nesse desafio mesmo que não possuam conhecimento em finanças.
  2. O torneio é acessível a ampla comunidade de cientistas de dados.
  3. A confidencialidade é resolvida ofuscando ou encriptando os dados de forma a preservar as propriedades matemáticas essencial para um problema preditivo.
  4. O overfitting é mitigado remunerando os pesquisadores através do desempenho fora da amostra. Além disso os pesquisados são incentivados a apostar o próprio dinheiro em suas previsões.

Fonte: DE PRADO, M. L., FABOZZI, F. J. “Crowdsourced Investment Research Through Tournaments”, The Journal of Financial Data Science, v. 2, n. 1, pp. 86–93, 2020.

Esse Torneio já existe¶

Modelos de torneio existentes¶

  1. Numerai Signals: Os competidores são livres para usar qualquer tipo de dados que dispuserem referentes a uma cesta de 5000 ações divulgadas de forma explícita. A partir dos próprios dados devem treinar um modelo preditivo capaz de prever o comportamento de cada dessas ações no futuro.
  2. Numerai Tournament: Essa é a competição principal, configurada exatamente da forma descrita por De Prado (2020), usando os dados encriptados disponibilizados pela equipe da numerai e as predições são feitas nos mesmas ações da Numerai Signals, porém anonimizados.

Adicionando Indicadores ao modelo¶

Agora que já temos os preços, precisamos configurar indicadores que adicionem valor preditivo ao modelo.

Índice de Força Relativa (RSI):¶

O RSI é um indicadorde análise técnica que busca medir se o ativo está excessivamente vendido (RSI>70) ou excessivamente comprado (RSI<30).

$$RSI = 100 - (100/(1 + RS))$$

RS = Média dos retornos nos períodos de alta / Média dos retornos nos períodos de baixa

Fonte: WILDER, J.New Concepts in Technical Trading Systems. Trend Research, 1978

Configurando o indicador¶

Modelos de Machine learning funcionam bem com dados distribuídos na forma de histograma. Aqui faremos uma divisão em 5 bins.

A Variável Alvo (Target)¶

  • A numerai é responsável por disponibilizar a variável alvo para cada ativo, para a próxima era. Uma era começa toda sexta-feira e será baseada nos retornos dos dias subsequentes.

  • A variável alvo é configurada como um problema de classificação, normalizadas e neutralizadas dos retornos dentro de cada era.

A distribuição do target¶

O target assume valores de {0, 0,25, 0,5, 0,75, 1}, mas as classes são desbalanceadas

  • 10% dos valores assumem 0 ou 1.
  • 40% dos valores assumem 0,25 ou 0,75.
  • 50% dos valores assumem 0,5.

Objetivos Gerais¶

  • Investigar a literatura de apredizado de máquina aplicado a finanças

Objetivos específicos¶

Muitas das aplicações aqui tratadas são vistas como uma disciplina a parte dentro da área de machine learning (De Prado, 2018), entre elas trataremos de:

  • Analisar modelos de ranqueamento
  • Investigar os principais métodos de validação de séries temporais
  • Desenvolver um critério para seleção de métricas
  • Desenvolver técnicas de combinação de modelos
  • Desenvolver modelos de híbridos de gradient boosting
  • Investigar os principais métodos de neutralização de variáveis
  • Revisar a literatura de seleção de variáveis
  • Desenvolver métodos clusterização de variáveis e séries temporais

Fonte: DE PRADO, M. L. Advances in Financial Machine Learning. 1ºed. Hoboken,New Jersey, Wiley Publishing, 2018.

Justificativa¶

De Pradro (2018), menciona que os fundos quantitativos mais bem sucedidos da indústria se baseiam nas técnicas de objeto de estudo deste trabalho, além disso, entende-se que esse estudo, apesar de se situar no âmbito financeiro, tem a oportunidade de explorar diversas técnicas de aprendizado de máquina que podem ser usadas em outras aplicações.
